This repository contains a reduced version of my Term Rewriting System (TRS) code that I used for Chapter 4 of my PhD Thesis. This is the version of the TRS that I used to conduct my experiments. It includes a jupyter notebook which can be used to calculate weights of any well-formed term, to allow users to see how the weights are calculated. The notebook will also show the user the normal form of any term and the corresponding weight vector. Termination and confluence have not been shown for this system.
